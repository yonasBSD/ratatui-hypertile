  # ratatui-hypertile
  
  Hyprland-inspired BSP tiling for Ratatui with no extra dependencies.
  Splits, focus, resize, movement, plugins, command palette, and tabs.
  
  ![demo](assets/demo.gif)
  
  ## Crates
  
  - **`ratatui-hypertile`** - core engine with full control
  - **`ratatui-hypertile-extras`** - runtime helpers with plugins, palette, keymaps, and tabs
